📱 FreezeTrack als App installieren

Installieren Sie FreezeTrack als PWA (Progressive Web App) für die beste Erfahrung

🎯 Vorteile der App-Installation:
  • ✅ Kamera-Berechtigung wird gespeichert - Kein wiederholtes Genehmigen
  • 📱 Startet wie eine native App vom Homescreen
  • 🚀 Schnellerer Start und bessere Performance
  • 📶 Funktioniert offline nach der ersten Nutzung
  • 🔔 Benachrichtigungen möglich (optional)
  • 🎨 Vollbild-Modus ohne Browser-UI

🤖 Android / Chrome

  1. Öffnen Sie FreezeTrack in Chrome
  2. Tippen Sie auf das 3-Punkte-Menü (⋮) oben rechts
  3. Wählen Sie "App installieren" oder "Zum Startbildschirm hinzufügen"
  4. Bestätigen Sie mit "Installieren"
  5. Die App erscheint als 🧊 FreezeTrack auf Ihrem Homescreen

🍎 iPhone / iPad (Safari)

  1. Öffnen Sie FreezeTrack in Safari
  2. Tippen Sie auf das Share-Icon (□↗) unten in der Mitte
  3. Scrollen Sie nach unten und wählen Sie "Zum Home-Bildschirm"
  4. Ändern Sie den Namen falls gewünscht: "FreezeTrack"
  5. Tippen Sie auf "Hinzufügen"
  6. Die App erscheint als 🧊 FreezeTrack auf Ihrem Homescreen
⚠️ iOS Hinweis:
Verwenden Sie Safari, nicht Chrome. Nur Safari unterstützt PWA-Installation auf iOS.

🖥️ Desktop (Chrome/Edge)

  1. Öffnen Sie FreezeTrack in Chrome oder Edge
  2. Schauen Sie in die Adresszeile rechts nach einem ⬇ Install-Icon
  3. Klicken Sie auf das Icon und dann "Installieren"
  4. Alternative: 3-Punkte-Menü → "App installieren"
  5. Die App startet in einem eigenen Fenster ohne Browser-UI

📷 Kamera-Berechtigung dauerhaft erlauben

  1. Nach der Installation öffnen Sie die installierte App
  2. Beim ersten Start: "Kamera erlauben" wählen
  3. Die Berechtigung wird dauerhaft gespeichert
  4. Alternative: Browser-Einstellungen → Website-Berechtigungen → Kamera: "Immer erlauben"

🌐 Alternative: Browser-Einstellungen

Falls keine App-Installation möglich ist:

  1. Klicken Sie auf das 🔒 Schloss-Symbol links neben der URL
  2. Wählen Sie "Website-Einstellungen"
  3. Setzen Sie Kamera auf "Zulassen"
  4. Die Einstellung wird für diese Website gespeichert
🚀 Nach der Installation:
  • Starten Sie FreezeTrack über das App-Icon
  • Die Kamera startet automatisch ohne Nachfrage
  • Alle Funktionen arbeiten wie eine native App
  • Updates erfolgen automatisch im Hintergrund
